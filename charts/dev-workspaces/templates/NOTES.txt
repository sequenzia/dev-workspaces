{{- if and .Values.oauthProxy.enabled .Values.oauthProxy.cookieSecret.existingSecret }}

 Suspend / Resume
------------------------------------------------------------------------------
  Suspend (scale to 0 replicas, retain PVC):
      helm upgrade {{ .Release.Name }} <chart> -n {{ .Release.Namespace }} --reuse-values --set suspended=true

  Resume (scale back to 1 replica, prior data intact):
      helm upgrade {{ .Release.Name }} <chart> -n {{ .Release.Namespace }} --reuse-values --set suspended=false

 PVC retention
------------------------------------------------------------------------------
  The PVC (volume "workspace-data") is annotated helm.sh/resource-policy: keep,
  so it SURVIVES "helm uninstall {{ .Release.Name }}". Your $HOME at
  {{ .Values.persistence.mountPath }} is preserved. To reattach it later, reinstall
  with the SAME release name ("{{ .Release.Name }}") and the same user.
